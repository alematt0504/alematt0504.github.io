Python
OWL
Enhancement of a dataset used for a generic Machine Learning task to improve its performance and metrics. First, an automatic mapping of the dataset information into a well-formed ontology (complying with specific standards) is performed. Then, SWRL rule inference is applied to deduce new information. OLIVE TREE SPECIES RECOGNITION USING MACHINE LEARNING TECHNIQUES
Matlab
Use of Machine Learning algorithms to accurately classify different types of olive trees from 47-band hyperspectral satellite images. A segmentation mask was used to separate the tree canopies from the ground. Then, for each canopy, the average spectral signature was computed. The resulting data enabled us to classify each individual tree using algorithms such as Random Forest and Singular Value Decomposition.
